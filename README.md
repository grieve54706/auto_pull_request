# auto_pull_request
Auto create pull request

### Script

create_pull_request.groovy

### Main

auto_pr.py

### Config

/resource/

### Steps

1. Find SIT latest tag
2. Find UAT latest branch
3. Create branch
4. Push branch
5. Find diff commit
6. Build pull request desc
7. Get pull request reviewers
8. Build pull request obj
9. Post to create pull request
